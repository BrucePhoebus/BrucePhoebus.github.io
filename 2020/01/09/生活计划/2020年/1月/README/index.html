    1月总结

    生活计划/ 2020年/ 2月 时感
    创建时间：2020-01-09 11:07
    更新时间：2020-02-07 15:20
    字数:1.6k 阅读:
    评论:
    1. 完成change过渡
    2. 找到了源动力
    3. 关注点转移
    4. 疫情关注

    完成change过渡

    • 自2019年12月开始的改变以来，生活波荡起伏，虽然彻底进行了自我反思，但是终究忽略了很多，曾经很自然的一些反应又变得刻意了。例如太刻意去跟朋友聊天，太刻意针对某些话题很激动，太刻意追求一些东西，生活一些状态失衡，一方面自己的生活混乱了，另一方面也引起了一些朋友不开心

    • 但是经过一个月的撞墙，很多东西都慢慢清晰了起来，道路也清晰起来，知道了要专注自己的生活，让自己沉淀，找到了下一步的长期计划，未来可期

    • 当然，从强烈的探索变成了稳步推进，已经找到了很多目标，很多期望，很多生活方式，现在需要的是持续消化，微调生活，让自己的生活更加有序，循序渐进，将大部分注意力从外部，转至内部成长

    但是change的伟力依旧让我难以忘怀，我希望在我停滞的时候能继续想起这个伟大的力量，让我的生活有持续的动力，犯错并不可怕，可怕的是不知道自己犯错

    找到了源动力

    • 实际上之前是靠自我激情推动生活，疯狂向未知领域探索，很多自己避让的问题和生活都进行了一部分探索，直到碰了很多墙才慢慢的冷却下来，虽然对别人来说这是乱七八糟的生活，或不成熟，但是对我来说，却是一个伟大的转至，去经历才懂得，很多东西别人说或看书是没用的，必须去碰撞，然后才能找到自己的路

    • 慢慢的从疯狂的寻找而碰撞中明晰了一部分生活，找到了自己想要的部分生活，是的，还是具有不确定性，因为还有些问题我并没有时间继续浪下去，但是基本道路找到了，有了长期计划，重点是计划是根据自己追求的生活落实的，明确了自己想要的生活和落实方式

    • 当然理想和现实是存在差异的，但是这个需要后续持续调整，但是整体道路是不能变的，也是能力范围之内的，对这个计划的实施有动力，也有强烈的学习兴趣，实现持续自我成长

    明晰了道理，自我驱动才是源动力，我们的生活，我们的理想需要我们脚踏实地的去践行，一步一步的去创造。

    关注点转移

    • 曾经我的关注点是让自己的生活更加快乐和活力，我把注意力给了我的亲人朋友，但是最后发现平淡才是生活，太刻意会造成他们的不适，我们每个人都有每个人的生活，或许只有伴侣才能和我们营造密切的关系，其他的一概不愿，因为他们也有他们的生活

    • 然后多次碰壁之后，我才慢慢的从外部转移到自身，也从哪些成功人士，或哪些踏实工作生活好人的身上看到，我们需要脚踏实地的去做自己想做的事，更多的是工作，是追求自己的事业，践行自我价值的实现，信仰等。也就是要专注自我实现，而这个自我实现更多的是我们的事业，其次是我们的生活，当然现在我这个年龄关注生活是难以自处的，我需要实现自我价值，还是要完成自我事业的构建，也就是专注工作

    • 所谓的专注工作，就是选择我想要走的道理，虽然我是一名程序猿，但是技术却不是我的追求，也太多危机感，还是努力考公务员，这个生活或许才是我追求的生活，安贫乐道，实现自我价值和社会价值，重点是有时间做自己想做的事。

    注意力从外部转至自身后，生活更是轻松踏实，知道了外部追求的虚无，自我实现才是理想结合实现是方式，去践行自我道理，让自己能明显看到自己的成长，不要将希望寄托于TA人。

    疫情关注

    • 当疫情来临的时候我总是相信政府能比较好的解决，毕竟有SASS这个前例，可惜中国的现状仍超乎我的相信，整个中国愿不及我理想的那么美好，美好的中国还需要我们一代又一代人的贡献

    • 还好还是有很多无私奉献的人，还有很多充满爱心的人，这个社会就是因为有这些有爱心的人，有这些贡献者，我们的社会才会变得越来越美好，我是看太多冷漠了，远离了党，看到了太多社会，被黑暗遮蔽了双眼，实际上我们这个世界需要我们去努力，去贡献。那些冷漠的人，就让他们过他们的小日子吧，我们将改造这个世界

    • 当然回归现实，湖北的境况还是很让我担忧，尤其是扩散的难以控制，还有很多自私无知的人，缺乏社会责任，到处传播病毒，这是很难想象的，尤其是现在年已过，多地正要开工大吉，却因为这些人导致疫情愈演愈烈，只是希望政府对这些人严惩，然后努力控制吧，不希望我们这些人也要上场，那时候真是大灾难。

    愿有社会责任心的人更多些，愿我们共同努力能尽早控制住这场灾祸，早控制，万事大吉，晚了，万事皆休。

    • 而我也准备去上班了，广州，倒是不知道有没有大巴出行，不知道有没有交通工具去广州，这事也是要看缘分，有也好，说明没那么严重，广州那边问过吃住都没问题，如果没有发车，那就难了，都好，希望政府能省点心，下一步应该是军管了吧。
