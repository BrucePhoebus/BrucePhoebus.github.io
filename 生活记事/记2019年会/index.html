    记不愉快的年会

    前言

    昨天晚上接受了年会晚会，虽然晚会后期氛围很不错，只是越到后面我的心越是苦涩，越是孤独，激情已尽。

    实际上晚会后面的表演也是出乎我的预料的，因为我们总是很少看到他们表演，尤其是平时在公司，我们几乎都是没看到多少积极的表演，所以对这方面也是吐槽不已，所以自己也缺乏努力，最后成功也只能说一分耕耘一分收获，因为自己付出得少，所以对自己的表演也不好说什么了。

    客观说说节目

    1. 主持人唱情歌。很不错的开场，有点气氛

    2. 舞蹈。有特色，反响也有，但是气氛一般。

    3. 小品。不熟练，切换不流畅，存在点停顿，而且控场放错背景音乐，尬场，没反响，没气氛

    4. 唱歌跳舞。流畅，但出现点问题，控场放错背景音乐，反响也一般，但有点小气氛，演员自行控场。

    5. 为你转身。有点尬，但是演员流畅演下去，少停顿，多小队表演，有特色，有些反响和气氛。

    6. 特色印度舞蹈。反响不错，氛围不错。

    7. 个人两首歌曲。演员控场，反响不错，氛围不错。

    8. …

    越到后面越有气氛，前期几个项目全部低沉，越到后面唱歌跳舞氛围越好，无关唱得多好，而是流畅，有点看头，支持，当然演员会渲染气氛，会带动全场也有，所以说会表演的还是很多，有看头的还是很多，唯一的问题是投入太少，总是感觉全都是靠氛围，上不了台面。

    然后说说吐槽的会场事项

    1. 布场太慢太简单，基本是踩点的，太匆忙了，而且布场很一般，一切从简，一眼就看出来没做什么准备，没投入什么。。。

    2. 控场太low，经常出错，导致有些表演出现卡顿，不流畅，效果打折扣

    3. 灯光控制，压根就没控制，全程亮灯，正常应该舞台亮灯就好，这样效果更好，结果全场亮灯，效果有差了一个档次

    4. PPT控制效果也一般，缺乏灵敏性，出现过几次意外，很简单的错误也犯了好几次，简单说就是low

    5. 非唱歌跳舞的控场都很差，效果很差，反响也很一般

    6. 吃的一般，很一般，而且菜色连十样菜都没有，哎，就是让人喝酒来的，当然，吃饱是没问题的。

    7. 没有对表演有足够的验收，部分表演效果比较差，还好大家本身不抱太多期望。

    当然也要赞一下好的地方

    1. 主持人很不错，很会控场，很会调气氛，也很会圆场，一大亮点

    2. 流程简单不冗余，当然这是因为节目节奏很快，而且穿插主持人圆场和很多抽奖环节，总体流程不错

    3. 凡是唱歌跳舞的都不错，因为就一首歌控场，很简单，而且反响很好，也就气氛不错

    4. 中后半场气氛足，我也努力当了啦啦队，尽力了

    总结

    1. 简单说整场因为气氛调动，还不错，总体收获了很多愉悦，大家都挺尽兴的，当然除了少部分不喝酒的。

    2. 缺乏投入。事前没有计划好，很多事情不合理，最后如何不是氛围调度好，这场就蹦了。

    3. 总结起来，没有实质的好，领导说了很多空话，也不展示啥好的，但是因为气氛调度起来了，多数人玩得还不错，半数人也获奖了，也是一个亮点，但是没获奖的就苦逼了，而我就是其中一位

    4. 这样的年会最好的就是唱歌跳舞，一体化，一首背景音乐。其他节目都很容易出问题，而相对比较简单的节目也就能更好的获奖，当然跳舞和乐器也不简单，但是单纯唱歌也能获得不错的奖项，而才艺表演却很难获得响应，当然，一分耕耘一分收获，自己演得不好也不怪别人。总之是控场. 背景音乐. 熟练度. 灯光等问题决定了，唱歌跳舞比较适合这样的舞台，其他不行，因为投入太少。

    5. 还是之前的想法，还不如请人来表演，或大家唱歌跳舞，大合唱，可惜我们总是希望特色和廉价，却不考虑最后效果，我们总以为总以为，却总是没想到最后结果，最后我们还是适合简单，因为我们本身就不打算高投入。

    6. 很高兴后面气氛调节得好，因为这样少了很多尬场，因为如果少了欢呼声，那一切都很尬，因为这个，很多都无所谓了。

    7. 如果有明年，如果不多点投入，我更愿意唱歌跳舞，而不是尬演
