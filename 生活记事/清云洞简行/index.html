    清云洞简行

    生活记事 爬山
    创建时间：2020-02-06 22:45
    更新时间：2020-02-07 00:03
    字数:2.6k 阅读:
    1. 前言
    2. 简单的启程
    3. 上山的路
    4. 行人三两个
    5. 另记
    6. 结束

    前言

    • 这是位闲的发慌的老友临时说要去的，无所事事太多天，总要找些事做，我倒是大把事，他兴趣也就出去玩

    • 不知道为什么选这个点，因为最熟悉就是清云洞了，可能也好久没去了吧，当然也是近，或简单

    简单的启程

    • 实际上我在出发前跟别人聊了回天，刚好隔壁老兄好久没见回来了，所以就聊上了，让老友等了下，罪过罪过，当然实际上我需要远程办公，也就是有事干，虽然没多少工作，但是自己还是安排了，但是索性大家回家没几天，工作让位生活

    • 带上口罩，开着女装摩托，到老友家就出发了，开车贼快，差不多5分钟就到山下了，然后找老地方放车，带瓶水就上山

    上山的路

    • 因为路修过，水泥路，所以爬起来比以前好多了，但是也比较陡峭，毕竟运动量少，爬几百米就气喘吁吁了，到中途就想要休息了

    • 在这个老地方爬山最好的地方就在于我们能回忆过往，看着周围熟悉或陌生的风景，总有说不尽的话题，但是或许我不应该穿太正式，如果外套是运动装就更能放开了，就能想以前，跑跑跳跳，气氛才更好

    • 记得以前还是土路的时候，就经常跑跑跳跳，中间就能玩，没有了水泥路风景更加自然，空气中更多的是泥土的味道，清新的空气，大自然的亲近，现在更多的是灰尘，是水泥路的感觉。当然我感觉建水泥路就是方便人上山，但是这样修的是水泥路，而不是修整坑坑洼洼，导致很多人直接开车上山，少了几分爬山乐趣，多了几分现代世俗的气息

    • 当然，我们并不关注太多世俗的问题，我们讨论周围的树木，那青梅树的花和果，聊着曾经摘青梅的日子，那些欢声笑语；说着那养蜂的箱子，那些蜂蜜，可惜没胆子去碰，当然也是说着笑，我们并没有这个兴趣；还有聊着这个水泥路啊，是给领导上山容易些呢？还是真了为了节省信众上山的辛苦(乐趣)，让更多人更方便的上山上香祈愿呢？反正很多人开车上去，然后上去就是在哪里往下看风景，喝着饮料，吃着零食，满地都是垃圾，就是不知道得偿不。

    • 偶尔回头往下看，才知道这爬山更多的乐趣，一览众山小，视野更宽广，心胸也开阔了，吸着新鲜空气，看着那一栋栋的民居，不由得思考，我们一辈子就为这些，值得？当然，我的答案是不值得，我们的生活不止这些，最终的目的是幸福

    • 没走多久就气喘吁吁了，真是四肢无力，头脑简单，哈哈哈，赶紧找个山坡坐休息，当然喜欢外出的老友就吐槽我们的体质，年纪轻轻，我们的确是该多运动了，我可不喜欢靠药瓶子吊命的生活

    • 找到熟悉的山坡，一块块大石头可以供我们休息，也有辆车停在这坡上，前面正有为若隐若现的老兄在哪石块上躺着，旁边还有个闲不住的金毛犬，应该就是车主人了，风景很好，位置也很好，甚至有土地公、土地婆在坐镇，可惜的是旁边太多垃圾，又是果皮，又是塑料袋，还有一个个饮料瓶，甚至偶尔还看到一两个口罩，可见是最近上来人丢的，甚是无语，我就在想啊，这种人是准备就上来一次？以后一辈子都不来了么？还是说对垃圾堆也有爬山的乐趣，不懂啊，甚是不懂。

    • 不过我们尽力忽略周围不和谐的环境，眺望远方，静静享受这海阔天空似的自然视野，吹着清凉的北风，看着远处渺小的建筑和绿绿苍苍的小山，很是惬意。不过值得一提的是，现在很多少越来越显得稀疏，或许是火烧山，或许是人为的砍伐，当然部分是村镇的扩容，但是总感觉缺了点意思，这好好的景色，总和我小时候的记忆不匹配，不知道有多少人意识到这样的不和谐，不知道全国各地又有多少这样的地方，和谐的自然不懂得保护，人造的自然总是缺乏一种自然的美。

    • 喘气久了，也说了不少话，三人就在大石头坡上喝着饮料，看着那些村落，说着曾经的故事，过去的总是过去了，失去的我们总会觉得是美好的，希望现在我们懂得珍惜当下，希望家乡越来越好，人与自然和谐相处

    • 时间总是匆匆流逝，饮料喝完，休息也差不多，只剩下一小截路，一路曾经是坑坑洼洼土路，形形色色，在水泥路的塑造下，马上全部变得一样，看上去就是上坡上坡再上坡，这也就是我讨厌现代建筑的原因，清一色的美在我眼中就变成了丑陋，没有多少兴趣去看，变了颜色和形状实际上也差不多，毕竟特殊设计实现的建筑太少，多是平庸的大众建筑

    • 很快就到了山顶，才发现这条山路2000米，花费了200万，换算了下价格差不多吧，只可惜这条路修得毫无特色，我开摩托也不敢上，还好有些人有这方面的兴趣，不然修出来也就意思意思。前方就是清云洞庙宇

    行人三两个

    • 实际上一路清云洞，还是有一些人的，但是因为当前疫情严重各地禁止活动的原因，清云洞庙宇也关门了，多数人是在家待久了显得发慌才出来爬爬山，可惜带口罩的终究是少出，我一行三人也就我戴口罩，主要是买不到口罩的原因吧，当然实际上我们本就不应该出门

    • 遇到行人我也是躲着走，关键时期还是大家注意安全，还好人不多，也就十来人，也没凑到一起，当然，或许除了我，也没人这么担心了

    • 清云洞山顶庙宇也因为关门甚是荒凉，也就两条看门狗在狂吠，我们就见见单单的看了一圈，实际上看头远不及眺望山下的风景，相比这些建筑，我更喜欢眺望远景或和人聊天，刚好有几个妹子过来，可惜我一见到女的就紧张，都不敢说话，赶紧绕过赶紧绕过，没啥看头就下山了

    回头的路上实际上也遇到了熟人，但是看他赶往山上，或许跟我们一样，可惜他们没有在山腰上停留更久，因为上去了估计很快就下山了，貌似人越来越多，我们也急匆匆的下去了，山下的景色看完，也就没啥看头了

    另记

    • 实际上下山的路上还关注到了一截被挖断的山崖上的建筑，长长的铁皮房紧靠崖边，不少地方都有突出，尤其是中间一端是靠一包包的沙袋支撑起来的，应该是曾经断层过，我看着都感觉危险，对住在这种房子的人感到无奈，可能人家花了大钱才有这样的房子，但是住在断崖上我真不知道他们是否能住得安稳，应该是住了不止一户人，但是我看着就觉得非常危险，不知道哪个夏天的暴风雨就把这断崖给冲没了，但是他们或许也是很无奈

    • 实际上不止这些人，很多人都住着危房，房子久了就会有些被压裂的问题出现，也就是房子老化，我自家的老房子也开始出现这样的问题，不过准备换新房的我并不介意，辞旧迎新，这就是生活

    希望那些人懂得改变吧，长期住在那样危房中总不是办法，我们失去的时候才懂得珍惜，那时总是太迟，希望我们都能多考虑一点

    结束

    • 我们匆匆的走，正如我们匆匆的来，可惜的是我不够激情，貌似我对这些都缺乏激情，当然也可能是穿着的原因，或者灰蒙蒙的天气代表了我缺乏激情的心，不过手机电脑看久了(或文字看多了)，也感觉看世界都有点晕晕的，调整生活，我并不喜欢这样的生活，期待未来的改变

    • 车来车往，我因为曾经摔过，现在开车贼稳，但是看着这些车，心里也发虚，一方面是担心病疫传染，都不知道为啥这么多人出门，都感觉不怕了；另一方面，看死的人多了，自己也就怕死了，还没看开，虽然总是说着不怕死，但是很多时候都是浮想联翩，莫名的胡思乱想

    • 回程也贼快，可惜没有妹子，本来要去老友上面喝茶的，不过他主动说回家，也好，回去工作，生活继续，或学习，这个关键的时间，宅得好。

    以后还是要多运动，每次看自己的身材、自己的体能、经常出现的腰酸背痛，对挂药瓶子的生活非常厌恶，同样厌恶城市的生活，还是喜欢农村悠闲自得的生活，有时间运动，或通过干活锻炼下，总是不能总坐着，身体是革命的本钱。
